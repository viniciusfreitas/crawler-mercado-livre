# Crawler para Mercado Livre

Crawler em Node.js para capturar informações de produtos na página do Mercado Livre

### Prerequisites

Node.js

## Running the tests

npm run test
npm run test:coverage


## Built With

* [Node.js](https://nodejs.org/en/docs/) - Chrome's V8 JavaScript engine
